Overview

Can COVID-19 policy choices predict which party a state's governor belongs to? This project trains machine learning models on 14 Oxford NPI policy features (school closures, mask mandates, vaccine policies, etc.) from 50 US states across 2020–2022, and evaluates how well policy alone classifies governor party (Democratic vs. Republican) over time.
